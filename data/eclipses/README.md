# Eclipse Dataset Structure

Generated by `tools/build-eclipses.js` covering 2000-01-01 to 2049-12-31.

```
data/eclipses/
├── solar.json        master index — 112 solar events (Total / Annular / Hybrid / Partial)
├── lunar.json        114 lunar events (Total / Partial / Penumbral)
├── bessel-poly.json  NASA Besselian polynomials for the four AE-direct events
└── events/
    └── <date>.json   per-event solar contact curves (limits, magnitude contours,
                       rise/set loops, P1–P4 anchors), one per solar event
```

`solar.json` / `lunar.json` carry only event metadata; the heavier per-event
contact-curve geometry lives in `events/<date>.json`, fetched lazily via each
event's `_curves_url` so the master index stays small. Real-time shadow geometry
(umbra/penumbra circles, central line) is not stored at all — it is computed
**client-side in `js/eclipse.js`** at every TimeState tick; see
`computeSolarShadow()` and `computeLunarVisibility()` for the ray-Earth
intersection and sub-lunar-point math respectively.

---

## solar.json — `Array<SolarEvent>`

```typescript
SolarEvent = {
  date:        "YYYY-MM-DD",
  kind:        "Total" | "Annular" | "Hybrid" | "Partial",
  peak: {
    time:      ISO_UTC,            // moment of greatest eclipse
    lat:       number,             // sub-shadow point latitude
    lng:       number              // sub-shadow point longitude
  },
  p1:          ISO_UTC,            // first external contact
  p4:          ISO_UTC,            // last external contact
  magnitude:   number,
  obscuration: number,             // 0..1
  gamma:       number,
  saros:       number | null,
  _curves_url: string              // "events/<date>.json" — contact curves, lazy-loaded
}
```

P1/P4 are computed by `Astronomy.SearchLocalSolarEclipse(date, observer)` at
the peak point — accurate to ~5–15 min vs. true global P1/P4. Falls back to
`peak ± 3h` if local search fails (rare).

## lunar.json — `Array<LunarEvent>`

```typescript
LunarEvent = {
  date:        "YYYY-MM-DD",
  kind:        "Total" | "Partial" | "Penumbral",
  times: {
    p1:        ISO_UTC,            // penumbra first contact
    u1:        ISO_UTC | null,     // umbra first contact (null for Penumbral)
    u2:        ISO_UTC | null,     // totality begins (null for non-Total)
    peak:      ISO_UTC,
    u3:        ISO_UTC | null,     // totality ends
    u4:        ISO_UTC | null,     // umbra last contact
    p4:        ISO_UTC             // penumbra last contact
  },
  magnitude:   number,
  obscuration: number,
  saros:       number | null
}
```
